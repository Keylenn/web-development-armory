高度和宽度

clientHeight、clientWidth的值由DIV内容的实际高度和CSS中的padding值决定

offsetHeight、offsetWidth的值由DIV内容的实际高度，CSS中的padding值，scrollbar的高度和DIV的border值决定

注意：CSS中的margin值，则不会影响clientHeight和offsetHeight的值
