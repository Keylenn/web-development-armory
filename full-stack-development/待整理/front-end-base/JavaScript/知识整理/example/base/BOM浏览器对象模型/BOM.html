什么叫BOM

1.概念

BOM，浏览器对象模型（Browser Object Model），可以对浏览器窗口进行访问和操作，BOM 有五大对象，包括BOM的核心--window对象,还有navigator 对象，screen 对象，history 对象，location 对象

2.易错解析

1）BOM 主要处理浏览器窗口和框架，不过通常浏览器特定的 JavaScript 扩展都被看做 BOM 的一部分

		这些扩展包括：
			弹出新的浏览器窗口
			移动、关闭浏览器窗口以及调整窗口大小
			提供 Web 浏览器详细信息的定位对象
			提供用户屏幕分辨率详细信息的屏幕对象
			对 cookie 的支持
			IE 扩展了 BOM，加入了 ActiveXObject 类，可以通过 JavaScript 实例化 ActiveX 对象

2）由于没有相关的 BOM 标准，每种浏览器都有自己的 BOM 实现。有一些事实上的标准，如具有一个窗口对象和一个导航对象，不过每种浏览器可以为这些对象或其他对象定义自己的属性和方法。

为什么要使用BOM

浏览器对象模型 (BOM) 使 JavaScript 有能力与浏览器“对话”，使用 BOM，开发者可以移动窗口、改变状态栏中的文本以及执行其他与页面内容不直接相关的动作。
